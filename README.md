# Task 2: Robotic Process Automation

### Dependencies

* Node.js 18 or higher
* npm

### Installing

* npx playwright install
* npm install

### Executing program

* How to run the program

Open the terminal
```
npm run scrapper
```

## Output

A CSV file will be generated with all the info from coinbase like `coinbase_2024-5-7_18-4-40`
